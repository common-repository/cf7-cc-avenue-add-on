Integrates CCAvenue with Contact Form 7 for redirecting user to payment page for the any purpose like donations or any other payment. 

== Description ==
= Overview =

This Addon integrates CCAvenue with Contact Form 7.

Each contact form can have its own CCAvenue settings. When a contact form is enabled with CCAvenue, and the user submits the form it will send the email as usual, then auto redirect to payment confirmation with details entered in plugin settings.  

If you have any problems, questions, or issues about this plugin then please create a support request and we will be more than happy to help

Note: This CCAvenue plugin works with both the old and new Contact Form 7 interface. A CCAvenue account is required to use CCAvenue integration. 

= Contact Form 7 - CCAvenue Integration Add-on Features =

*	Set Payment Title, Description and Amount per contact form
*   Built in support 25 currencies (Please confirm if your country has CCAvenue payment gateway. For any issue please contact to plugin support)
*	CCAvenue testing through SandBox
*	Choose a cancel payment URL
*	Choose a succesful payment URL

== Installation ==

= Automatic Installation =
> 1. Sign in to your WordPress site as an administrator.
> 2. In the main menu go to Plugins -> Add New.
> 3. Search for Contact Form 7 - CCAvenue Add-on and click install.
> 4. That's it. You are now ready to start accepting CCAvenue payment on your website through your contact form.

== Frequently Asked Questions ==

= How do I use it? = 

You can integrate CCavenue payment gateway with Contact Form 7 plugin. If configured properly, form will redirected to CCAvenue payment gateway with amount you have configured in the settings. 

= How do I get help? =

Help is provided via the [plugin support forum](https://wordpress.org/support/plugin/cf7-cc-avenue-add-on) only.

= How do I get Merchant ID, Working Key and Access Code  =

1. You can find your Merchant account ID in your CC Avenue Dashboard [https://dashboard.ccavenue.com/jsp/merchant/merchantLogin.jsp]
2. For any query you can check CCAvenue FAQ here in https://www.ccavenue.com/servicedetails_ccav.jsp
